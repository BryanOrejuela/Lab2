CÓDIGOS

El cubo led se ve como una matriz de leds, empezamos creando un bucle principal en el código que se ejecuta continuamente. Dentro de este bucle, se actualiza la matriz de datos de acuerdo con el patrón o secuencia que deseamos.

ARMADO DE MAQUETA

El armado de nuestra maqueta fue un proceso para representar de una manera tridimensional nuestro cubo led y de ese modo queremos mostrar cómo será la forma final de nuestro producto.

ARMADO DIGITAL

El programa 3D Max implicó la creación del modelo tridimensional de nuestro cubo led, la aplicación de medidas, la configuración de la iluminación y la adición de animaciones. El resultado final es una representación digital realista y visualmente atractiva de nuestro cubo led Aura.

IMPRESIÓN

El proceso de nuestra impresión es para armar la estructura de nuestro cubo y así tener una base compleja y sólida con nuestras medidas ya establecidas.

BRANDING

Aura es un cubo con una escala de colores entre morados y naranjas los cuales capturan la esencia del proyecto de manera llamativa y memorable. Representa la fusión entre la innovación tecnológica y la expresión artística, proporcionando una experiencia visual impactante y cautivadora para quienes se encuentren con el cubo LED Aura.

ARMADO DEL CUBO

Luego de la impresión de cada pieza de nuestro cubo led, empezamos con el armado, soldar las luces y las otras partes de nuestro cubo led.
